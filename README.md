Clone do Catálogo Netflix / Netflix Catalog Clone

Sobre o projeto / About the project

Português:Este é um clone simples do catálogo da Netflix, desenvolvido apenas com HTML, CSS e JavaScript puro. O objetivo do projeto é praticar conceitos básicos de desenvolvimento front-end, como estruturação de páginas, estilização responsiva e manipulação de elementos com JavaScript.

English:This is a simple clone of the Netflix catalog, built only with HTML, CSS, and pure JavaScript. The goal of this project is to practice basic front-end development concepts such as page structure, responsive design, and DOM manipulation with JavaScript.

Funcionalidades / Features

Português:

Layout inspirado na interface da Netflix

Carrossel de filmes e séries (rolagem horizontal)

Efeito hover nas capas dos títulos

Menu fixo superior com logo e navegação

Design responsivo (adaptável a celular, tablet e desktop)

English:

Layout inspired by Netflix's interface

Movies and series carousel (horizontal scrolling)

Hover effect on covers

Fixed top menu with logo and navigation

Responsive design (works on mobile, tablet, and desktop)

Tecnologias utilizadas / Technologies used

HTML5 – Estrutura das páginas / Page structure

CSS3 – Estilização e responsividade / Styling and responsiveness

JavaScript – Interações e funcionalidades dinâmicas / Interactivity and dynamic features
